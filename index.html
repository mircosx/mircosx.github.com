Busqueda de un campo con filtracion por criterio, dinamico para cualquier campo/tabla 

<?php
function buscaCampo1Criterio($nombre_campo, $tabla, $nombre_criterio, $criterio) {
    $query = "SELECT $nombre_campo FROM $tabla WHERE $nombre_criterio='$criterio'";
    $resultado = mysql_query($query);
    $linea = mysql_fetch_row($resultado);
    $campo = $linea[0];
    return $campo;
}
?>

<?php
function buscaCampo2Criterios($scampo, $fcampo, $nombre_campo1, $criterio1, $nombre_campo2, $criterio2) {

Funcion que rellena automaticamente los OPTION de un SELECT de un FORM cargando los datos desde una base de datos 

<?php
function rellenar_N_Campos_Dinamico($ftabla, $orden) {
    $query = "SELECT * FROM $ftabla ORDER BY $orden";
    $result = mysql_query($query);
    $contcol = mysql_num_fields($result);
    echo "<SELECT NAME='$ftabla'>";
    while ($col = mysql_fetch_row($result)) {
        echo "<OPTION VALUE='$col[0]'>";
        for ($i = 1; $i < $contcol; $i++) {
            echo $col[$i] . " ";
        }
        echo "(" . $col[0] . ")</OPTION>\n";
    }
    echo "</SELECT>";
}
?>
